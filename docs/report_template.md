# Отчет по лабораторной работе №1

## 1. Цель работы

Освоение одношаговых методов численного интегрирования задачи Коши для ОДУ с элементами оценки погрешности на шаге и управления шагом.

## 2. Постановка задач

### 2.1 Тестовая задача

\[
\frac{du}{dx} = (-1)^{n}\frac{n}{2}u, \quad u(0)=u_0, \quad n=25
\]

Точное решение:

\[
u(x)=u_0 e^{-12.5x}
\]

### 2.2 Основная задача

\[
m u'' + c u' + k u + k^*u^3 = 0
\]

Начальные условия и параметры:
- `u(0)=10 см`, `u'(0)=0`
- `k=2.0 Н/см`
- `k*=2.0 Н/см^3`
- `c=0.15 Н·с/см^2`
- `m=0.01 Н·с^2/см`

Интервал интегрирования: `0 <= x <= b`, где `b >= 1 с`.

## 3. Метод решения

Используется метод Рунге–Кутта 4-го порядка:
- без контроля ЛП (постоянный шаг);
- с контролем ЛП (адаптивный шаг, параметр `eps`).

Оценка локальной погрешности на шаге:

\[
\text{ОЛП} \approx \frac{\|v_{h/2}-v_h\|}{2^4-1}
\]

Условия изменения шага:
- если `ОЛП > eps` => `h := h/2`, `C1++`;
- если `ОЛП < eps/16` => `h := 2h`, `C2++`.

Ограничения:
- счетчик шагов `i <= Nmax`;
- контроль выхода на правую границу `b`.
